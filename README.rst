-----------
Description
-----------
-----------------------------------------------------------------
Techhnologies : Python, bottle, HTML, JQuery, AJAX , D3.js, Docker
------------------------------------------------------------------
SentiLit is a Web based application where end user is provided option to upload any text file pertaining to literature. The application performs a sentiment analysis based on the word sysnsets generated by Senti word net for each word inside the given text file and generates the variation of the sentiment graph word to word across the text file . Further it also generates the number of positive, negative and objective word scores and counter and visualize them on the bar charts.
